Windows 系统下基于 Hexo 框架的博客构建

其·零 前言（碎碎念）

大一的时候学校没有安排什么专业课，自己自学也没什么动力，就这么混到了大二。到了大二，发现很多东西自己都没有学过，有些后悔。希望通过搭建这个博客来记录自己的学习过程，也给自己一点激励和鞭策的作用吧。

其·壹 为什么选择 Hexo

博客平台有很多，比如 Typecho 、WordPress 等等。相对于 Typecho 和 WordPress 这类动态博客来说，Hexo 跟他们的最大区别就是静态了。这里的“动态、静态”主要是指其对数据处理的方式，通过数据库来动态的查询、调用、展示的叫“动态”，直接生成纯.html网页文件来供浏览的叫“静态”。详请参考个人博客如何选择 Typecho 和 Hexo？-明月登楼的文章-知乎

Hexo 正常来说，不需要部署到我们的服务器上，我们的服务器上保存的，其实是基于在 Hexo 通过 Markdown 编写的文章，然后 Hexo 帮我们生成静态的 html 页面，然后，将生成的 html 上传到我们的服务器。也就是说 Hexo是一套使用第三方主题模板+自己的博客文章（.md文件）+网站信息配置，通过Node.js进行本地连接混合生成一套静态网站的工具链套件。简而言之：Hexo 是个静态页面生成、上传的工具。

作为一个轻量化的个人博客，Hexo即可满足绝大部分需求。而且 GitHub 的 Page 服务给用户提供了一个很方便的近乎零成本的部署方案。

其·贰 详细过程

这篇博客的整个流程主要是按照 B 站 up 主 CodeSheep 的视频 手把手教你从0开始搭建自己的个人博客 |无坑版视频教程| hexo 来配置的，不过由于我是 Windows 系统，所以有一些不同之处。如果你是 Mac OS 系统，可以移步链接处按视频照做即可。感谢 up 主和视频弹幕的无私分享~

安装 Git

Git 是一个开源的分布式版本控制系统，可以有效、高速地处理从很小到非常大的项目版本管理。它与 SVN，CVS 等版本控制系统最核心的区别就在于它是分布式的。知名的全球最大男性交友网站 GitHub 的核心部分版本控制就是用 Git 来处理的。

安装地址：https://git-scm.com/


直接在首页下载，或者进下载页面选择合适的版本下载。

Windows 系统的详细安装步骤请参考这篇教程：

https://www.cnblogs.com/xueweisuoyong/archive/2019/11/22/11914045.html

安装 Node.js

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。Node 是一个让 JavaScript 运行在服务端的开发平台，它让 JavaScript 成为与PHP、Python、Perl、Ruby 等服务端语言平起平坐的脚本语言。有一种说法说它就是一套SDK，一套使用JavaScript编写的，包含多个应用级功能模块，可用于一般应用开发的工具包。

安装地址：https://nodejs.org/en/

选择LTS（Long Term Support）长期支持。安装时一路 next 即可。

Win10 Cortana 搜索栏直接输入 cmd 后（不要立即回车进入），按住 CTRL + SHIFT + ENTER 键可直接进入管理员模式命令行。在命令行中输入 node -v 与
npm -v 来查看版本号，从而验证安装成功。

安装 cnpm

由于 npm 国内镜像源的安装速度很慢，所以先利用 npm 来安装 cnpm（阿里巴巴的一个开源镜像）。

在命令行中输入:

install -registry=https://registry.npm.taobao.org

安装 cnpm
安装完成

安装完成后，也可以输入 cnpm -v 来验证安装完成。

安装 Hexo 博客框架

在命令行中输入：

cnpm install -g hexo-cli

安装完成

安装完成后，也可以输入 hexo -v 来验证安装完成。
